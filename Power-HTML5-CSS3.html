See HTML5 and CSS3 in action

If you're using a modern browser you should be able to see a pretty cool animation below. It is completely crafted using only HTML5 and CSS3 and it's rendered using no javascript, no Flash, no plug-ins of any sort. Here's to standards that raise performance and efficiency!

While I like to think I know a good thing when I see it, I also know that my coding skills are not advanced enough to create this type of animation. So, as I bring it to you I also salute the creator, Robin Delaporte, who chose to make his code reusable under the terms of the MIT license, and the standards board that made a power set of standards available for us all.
